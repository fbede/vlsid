# Variable Length Sortable ID (VLSID)
This is a variable length sortable id generator. It is designed to be like
nanoid but sortable.

## Features
- Generate sortable string ids.
- Ids can be anylength (minimum of 9 characters).
- Uses base64url characters, so ids are url safe.
